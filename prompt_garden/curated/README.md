# Curated Prompt Baseline

This folder contains the small tracked Prompt Garden baseline that is intentionally kept in Git.

Purpose:

- preserve the current best public prompt examples
- keep the main bot baseline understandable without exposing the full runtime workspace
- provide a small prompt set that can be evolved deliberately in future iterations
