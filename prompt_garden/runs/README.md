# Prompt Garden Runs

This directory is the execution-output zone for Prompt Garden.

It is split into:

- `raw/` for faithful model execution records
- `normalized/` for review-friendly answer records

Git policy:

- placeholder documentation is tracked
- real run artifacts remain local or generated by default

The detailed file contract is defined in `docs/prompt-garden-contracts.md`.

The current review-loading helpers for these artifacts live in:

- `src/chemistry_bot/promptops/review_store.py`
